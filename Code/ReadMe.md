##### 2.4.5 位运算
5种位运算：与、或、异或、左移、右移  
左移运算符：m<<n表示把m左移n位，左边丢弃，右边补0  
右移：如果数字是一个无符号数值，用0补；如果是有符号数值，用符号位补
